How To Play

Reverse Cells is a game that is played on a square grid. Each cell can be in two states: ON (black background) and OFF (white background).

This is a ON cell.

This is a OFF cell.

A grid is solved if every cell on it is OFF. To achieve this result, the player must make a sequence of moves. A move is done by touching a cell, effectively flipping the cell and all its horizontal, vertical and diagonal neighbors. Below there is an example of move:

When we touch the highlighted cell...

it gets flipped along with its neighbors!

If a cell is adjacent to a border, only its effective neighbors are flipped. The grid is not toroidal.
